# Simple CRUD with API

  Technologies Used: 
	  - React JS for front end.
	  - Node JS for back end.
	  - Express JS for API
	  - Knexjs as query builder
	  - mysql as database

## How to use

### Clone this repository

    npm install or yarn install

then run it.

    npm start or yarn start

server running on port 3000!
